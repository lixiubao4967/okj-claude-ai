# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Overview

This is a **documentation-only** repository containing Chinese-language (中文) guides for Claude Code usage. There is no application source code, build system, or test suite.

## Content

- `README.md` — The main document covering Claude Code installation, common commands, keyboard shortcuts, and Channels (Telegram/Discord) configuration. **[自编]**
- `gstack-install.md` — gstack 安装指南。**[自编]**
- `claude-code-plugins-guide.md` — Claude Code 官方插件指南，包含插件分类、推荐插件的安装步骤和使用方法。**[第三方文章解读]** 原文来自 DevelopersIO（Classmethod），基于 Anthropic 官方插件文档整理。

## Content Sources

- **自编**：由本仓库维护者原创编写。
- **官方**：直接翻译或搬运自 Anthropic 官方文档。
- **第三方文章解读**：基于第三方技术博客文章，结合官方文档整理而成，非 Anthropic 官方出品。

## Writing Conventions

- All documentation is written in **Chinese (简体中文)**.
- Commit messages use the `docs:` prefix (e.g., `docs: 添加 Claude Code 安装与使用说明`).
- Technical terms, command names, and code snippets remain in English within Chinese text.
