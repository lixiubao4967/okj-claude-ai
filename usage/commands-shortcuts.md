# 常用命令与快捷键速查表

## 核心命令（必须会）

| 命令 | 说明 |
|------|------|
| `claude` | 在当前目录启动交互式会话 |
| `claude "你的问题"` | 直接执行单次任务 |
| `claude --help` | 查看帮助 |
| `/init` | 初始化 CLAUDE.md，新项目第一件事 |
| `/compact` | 压缩上下文，长对话续命神器 ⭐ |
| `/model` | 切换模型（简单任务用 haiku，硬骨头用 opus）⭐ |
| `/clear` | 清空对话重新来 |
| `/resume` | 恢复之前的对话 |
| `/memory` | 编辑记忆文件，实现跨会话记忆 |
| `/diff` | 查看会话内的代码改动，提交前必看 |
| `/help` | 查看当前版本所有可用命令（命令会随版本变化） |
| `/add-dir` | 添加工作目录 |
| `/mcp` | 管理 MCP 服务器 |
| `/export` | 导出对话到文件 |
| `/vim` | 切换 Vim 模式 |

## 进阶命令（按需使用）

| 命令 | 说明 |
|------|------|
| `/cost` | 查看当前会话花费 |
| `/context` | 可视化上下文使用情况 |
| `/fast` | 切换 Opus 4.6 极速模式（仅 Opus 4.6 可用） |
| `/review` | 智能代码审查 |
| `/simplify` | 代码简化与质量审查（需安装 `pr-review-toolkit` 插件） |
| `/agents` | 创建和管理智能体 |
| `/hooks` | 管理工具事件钩子 |
| `/config` | 打开配置面板 |
| `/doctor` | 诊断安装问题 |
| `/status` | 查看版本、模型、账户状态 |
| `/security-review` | 安全审查待提交代码 |
| `/pr-comments` | 获取 GitHub PR 评论 |

## 快捷键（效率翻倍）

| 快捷键 | 说明 |
|--------|------|
| `Shift+Tab` | 在 normal / auto-accept / plan 三种模式之间循环切换 ⭐ |
| `ESC` | 中断执行，跑偏了立刻刹车 |
| `ESC+ESC` | 跳转到之前的消息 |
| `Cmd+K` | 打开命令面板 |
| `Cmd+B` | 后台执行命令 |
| `Cmd+O` | 展开/折叠过程信息 |
| `@文件名` | 引用文件 |
| `!命令` | 直接执行 bash 命令 |
| `#信息` | 写入 CLAUDE.md 记忆 |
| `Control+V` | 粘贴图片 |

## CLI 启动标志（在终端启动时使用）

| 标志 | 说明 |
|------|------|
| `claude --print "..."` | 执行单次查询后退出，适合写脚本 |
| `claude -c` | 接续当前目录上一次的会话 |
| `claude --dangerously-skip-permissions` | ⚠️ 跳过所有权限确认，仅在受信任的容器 / CI 环境中使用 |

> **效率组合推荐**：光会 `/compact` + `/model` + `Shift+Tab` 这三个组合，独立开发效率就能甩开大部分人。
